# Documentation Library

This folder is the dedicated documentation library for the project. Keep all
new guides, operational runbooks, and architecture notes here.

## Index

- docs/PLAN.md: master plan for MQTT admin platform
- docs/mqtt/README.md: MQTT control and telemetry overview
- docs/mqtt/EXAMPLES.md: MQTT usage examples and test commands
- docs/setup/BROKER-MAC.md: Mosquitto broker on Mac (Docker)
- docs/setup/ADMIN-PLATFORM.md: FastAPI + React admin platform (Docker)
- docs/setup/NODERED-PI.md: Node-RED MQTT flows and scripts on the Pi

## Conventions

- Use ASCII-only text to avoid encoding issues on devices.
- Prefer short, task-focused guides.
- Keep example topics and payloads in code blocks.
- Add links to related docs in the same folder.
